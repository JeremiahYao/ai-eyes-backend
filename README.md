This repository contains the backend design and AI processing logic for **AI Eyes**,
an assistive navigation system for people who are blind or visually impaired.

## Project Overview
AI Eyes uses camera-based computer vision and AI to:
- Detect nearby objects and obstacles
- Estimate distance using stereo vision
- Provide safe navigation feedback
- Integrate with Google Maps for directions

## Development Approach
- Backend logic is designed and tested online using Google Colab
- Code is version-controlled using GitHub
- Final deployment will be done on a stronger CPU/GPU system

## Security
- No API keys are stored in this repository
- All sensitive credentials are handled via environment variables
